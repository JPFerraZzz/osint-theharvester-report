# Investigação OSINT — Reconhecimento Passivo com theHarvester

Relatório técnico de reconhecimento passivo utilizando OSINT, desenvolvido por interesse pessoal no âmbito do curso de Redes e Sistemas Informáticos (ISPGAYA).

## Objetivo

Demonstrar o que é possível descobrir sobre organizações e infraestruturas utilizando exclusivamente fontes públicas e abertas, sem qualquer acesso não autorizado.

## Domínios Investigados

| Domínio | Tipo |
|---|---|
| nasa.gov | Agência governamental |
| rapid7.com | Empresa de cibersegurança |
| lagplace.xyz | Servidor pessoal (com autorização do proprietário) |

## Ferramenta Principal

- **theHarvester** v4.10.1 — reconhecimento passivo de emails, subdomínios, IPs e URLs

## Algumas Descobertas

- Emails e subdomínios internos da NASA expostos publicamente
- IPs reais dos scanners InsightVM da Rapid7 identificados via OSINT passivo
- IP real de um servidor doméstico exposto apesar da proteção Cloudflare ativa
